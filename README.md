## 說明
快速填表產生單表式後台 - 套版用
1. / - 基礎設施
2. /libraries - 共用函式庫
3. /framework - 共用框架
4. /code - 共用代碼
5. /project/backend - 單表式後台

## 使用套件
1. laravel - 主體 
2. hahahalib - 函式庫 
3. doctrine - 功能包 

備註 : 不用laravel doctrine，因為更新太慢，維護不方便

## batch_file腳本呼叫GPL程式問題
License是MIT，應該沒被GPL直接感染

但會出現GPL感染判斷問題，需要提供相關Code(batch_file) 
因為專案本身有開源，請自行到
https://github.com/hahaha0417/laravel_single_table_backend 
找對應Code


## GPL 間接感染問題
如有修改相關呼叫GPL的部分，請自行將GPL相關(前端，後端，函式庫)獨立資料夾，方便法院要求檢查時提供相關代碼
我用到的部分就是Aid(輔助)相關資料夾(controller，view，batch_file)

個人覺得只是間接感染，有GPL認定問題，要提供相關代碼 
基本上寫個小批次檔，將相關代碼打包交給法院即可(應該)
