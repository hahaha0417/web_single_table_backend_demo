## 方針 
1. 單表
2. 快速
3. 低成本

## 說明
快速填表產生單表式後台 - 套版用
1. / - 基礎設施
2. /libraries - 共用函式庫
3. /framework - 共用框架
4. /code - 共用代碼
5. /project/backend - 單表式後台

## 使用套件
1. laravel - 主體
2. hahahalib - 函式庫
3. doctrine - 功能包 

備註 : 不用laravel doctrine，因為更新太慢，維護不方便
